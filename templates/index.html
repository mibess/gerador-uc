Sobre o Gerador de UC (Padrão Aneel)

O Gerador de Código UC é uma ferramenta essencial para desenvolvedores, analistas de QA e testadores de software que trabalham com sistemas de energia elétrica. Nossa ferramenta cria números válidos de Unidade Consumidora (UC) respeitando o algoritmo de módulo e os prefixos das principais distribuidoras do Brasil.

O que é uma Unidade Consumidora (UC)?

Segundo a regulação da Aneel (Agência Nacional de Energia Elétrica), a UC é o conjunto de instalações e equipamentos elétricos caracterizados pelo recebimento de energia em um só ponto de entrega. Cada residência ou comércio possui um código único de identificação.

Para que serve este gerador?

Este sistema é voltado para fins de testes de software (Mocks). Ele permite simular cadastros em sistemas de gestão, faturamento e CRM de companhias elétricas sem utilizar dados reais de clientes, garantindo conformidade com a LGPD.

Distribuidoras Suportadas

O algoritmo suporta a geração de códigos válidos para diversas concessionárias, incluindo Enel, CPFL, Cemig, Light, Neoenergia, Equatorial, entre outras listadas no padrão nacional.
